CI Start Project


Download

Perkenalan
ci-start-project adalah sebuah tool untuk memulai aplikasi yang akan dibuat.
Keunggulan
  • Template Library
  • Web Settings (ex : project title)
  • GULP (kamu harus install gulp terlebih dahulu)
  • Simple Code
  • Pitisan
  • CRUD Generator by Harviacode (https://bitbucket.org/harviacode/codeigniter-crud-generator)
  • HMVC

Yuk Mulai!

  • 1. Download terlebih dahulu ci-start-project kemudian simpan di folder projek anda.
  • 2. Buka file autoload.php di folder config/autoload.php 

    $autoload['libraries'] = array();
ubah menjadi
$autoload['libraries'] = array('template');

$autoload['helper'] = array();
ubah menjadi
$autoload['helper'] = array('url','my');

$autoload['config'] = array();
ubah menjadi
$autoload['config'] = array('site_config');

  • 3. Buka file config.php di folder config/config.php 

    $config['base_url'] = '';
ubah menjadi
$config['base_url'] = 'http://localhost/ci-start-project/';	// sesuaikan dengan lokasi folder projek anda

  • 4. Buka file routes.php di folder config/routes.php 

    Tambahkan baris code berikut
$route['default_controller'] = 'home';
$route['profile'] = 'home/profile'; // ini hanya sebagai contoh merender page profile
$route['about'] = 'home/about';	// ini hanya sebagai contoh merender page about

  • 5. Untuk merubah title web anda, silahkan ubah file site_config.php di folder config/site_config.php 

    $config['site_name'] = 'Your Project'; // ubah nama projek kamu
$config['site_title'] = "Your Project";	 // ubah title projek kamu

  • 6. Sampai langkah ini ci-start-project sudah dapat di gunakan. Selanjutnya saya akan menjelaskan fitur yang ada lainnya
  • 7. Theme template

    Untuk mengubah theme atau struktur layout anda dapat merubahnya disini:
-libraries/Frontend_Controller.php
-themes/default_theme/views/layouts/default.php //template index
-views/partials/ //partials include (header,footer,navigation)

  • 8. Controller

    class Home extends Frontend_Controller {

	public function __construct()
	{
		parent::__construct();
		$this->theme();		
	}

	public function index(){
		$data = array();

		web_title('Home'); // set title web
		web_build('home', $data); // build view
	}
}

  • 9. Pitisan https://github.com/piece601/Codeigniter-Pitisan (optional)
  • 10. Gulp (Anda harus menginstal gulp terlebih dahulu) 

    untuk menginstall nya menggunakan cmd/terminal
1. masuk ke directory project anda
2. lakukan perintah npm install --save-dev gulp gulp-concat
*untuk lebih lengkapnya silahkan kunjungin situs gulp

  • 11. Simple Send Email 

    code yang sangat simple untuk mengirim email adalah berikut :
send_email($to, $subject, $data, 'file');

untuk settingan nya terdapat di file helpers/my_helper.php (silahkan explore lagi)
